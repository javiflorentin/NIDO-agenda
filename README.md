NIDO - Agenda
Este proyecto tiene como finalidad visibilizar la importancia de contar con herramientas que acompañen y faciliten las tareas de crianza y cuidado infantil. En muchas ocasiones, madres, padres y tutores deben administrar una gran cantidad de información relacionada con la salud, el desarrollo y la organización cotidiana de los niños, lo que puede generar dificultades para mantener un seguimiento ordenado y accesible. 

Como ejecutar
Se debe ejecutar:
            python main.py
para la apertura del programa

Estructura

main.py             
modules/
    agenda_panel.py
    config.py
    datos_panel.py
    informe_panel.py
    informe_pdf.py
logo_nido.png        
logo_unpilar.png     
requirements.txt     

Para la instalacion de este programa se debe clonar el repositorio: "gh repo clone javiflorentin/NIDO-agenda "con git clone en una carpeta nueva, una vez realizado esto, se puede ejecutar el programa.
